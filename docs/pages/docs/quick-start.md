# 快速开始

## npm 安装

```bash
npm i element-extension
# or 
pnpm add element-extension
yarn add element-extension
```

## 在项目中引入

```ts
// 指定组件
import Vue from 'vue'
import { Form } from 'element-extension'

Vue.use(Form)

// 全局引入
import Vue from 'vue'
import ElementPart from 'element-extension'

Vue.use(ElementPart)
```
